HTML

  • The head element contains information about the webpage (the metadata).
  • The body element represents the visible content shown to the user.
  • The meta element is information on the page that isn't visible but is used by the browser.
  • A semantic element means the name of the element describes its content and the role it plays.
  • Semantic elements make a page easier to read.
  • Div creates a division on the page.
  • Span creates an inline division.
  • An element is everything inside of the code from the opening tag to the closing tag.
  • The Section goes inside of the Body of HMTL code.
  • ID specifies the unique id for HTML.

Git

  • git status: checks out what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: lists your branches. a * will appear next to the currently active branch
  • git commit -m: commits staged content as a new commit snapshot
  • git stash: saves modified and staged changes
